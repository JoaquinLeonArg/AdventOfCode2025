# Advent of Code 2025
My solutions for the Advent Of Code 2025 challenges.

## Requirements
- Rust

## To run

### Running a specific day
```bash
cargo run <day> <input_file>
```

For example:
```bash
cargo run 1 ./input/day1.txt
```

### Running tests
Each day has a test suite that runs the example inputs given on the problem descriptions.
To run all of them, simply execute:
```bash
cargo test
```
